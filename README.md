# Mini C-like Compiler in Python

This project is a simple compiler that supports:
- Variables
- Integer arithmetic
- Operator precedence
- Parentheses
- Unary minus
- print() statement

Built using:
- Lexer
- Parser (recursive descent)
- AST
- Bytecode generator
- Stack-based VM

## Run
python main.py "cd: example1/example2.mc"
